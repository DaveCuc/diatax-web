# Security Policy

## Supported Versions

Only the latest stable version of **DiataxWeb** receives security updates. If a vulnerability is found in an older version, please upgrade to the latest release.

| Version | Supported |
| ------- | --------- |
| >= 1.0.0| :white_check_mark: Active |
| < 1.0.0 | :x: Unsupported |

## Reporting a Vulnerability

We take the security of **DiataxWeb** very seriously. If you find a security vulnerability, please do **NOT** open a public issue on GitHub. Instead, report it privately to the development team.

### How to Report

Please send an email to the maintainers of the **nosCONSUMIOLaIA** team with a detailed description of the vulnerability, including:

1. **Steps to Reproduce**: Detailed steps or a Proof of Concept (PoC) script.
2. **Impact**: What could an attacker achieve by exploiting this vulnerability?
3. **Environment**: OS, dependencies, and configuration details where the issue was found.

We will acknowledge your report within 48 hours and work with you to coordinate a secure release patch before public disclosure.
